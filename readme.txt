Create large numbers of posts on a single screen ( including Custom Post Types and Statuses )

== Description ==

Create large numbers of posts on a single screen in bulk. Features include:

 - TinyMCE visual editors
 - Custom Post type support
 - Custom post status support
 - AJAX powered
 - Validates against Coding standards
 - Checks Capabilities and Nonces for security
 - Filters for adding and saving extra fields
